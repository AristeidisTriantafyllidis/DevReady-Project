## ☎️ Call Center inspired by [Aircall](https://aircall.io/).
The call management interface provides a centralized, fully responsive platform for monitoring communication data. Users can browse real-time call logs, inspect detailed activity metrics, and securely archive historical records to ensure regulatory compliance and efficient data retention.

### 💻 Local Development

Follow these simple steps to get the development environment up and running on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AristeidisTriantafyllidis/DevReady-Project.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd dev-ready-project
   ```

3. **Install the required dependencies:**
   ```bash
   npm install
   ```

4. **Start the local development server:**
   ```bash
   npm start
   ```
   _The app will automatically open in your browser at `http://localhost:3000`._

5. **Run the test suite (Jest):**
   ```bash
   npm test
   ```
