Call Center inspired by [Aircall](https://aircall.io/).
View, browse, and archive call activity from a clean, responsive interface.

Tech Stack

React 19-component-based Ui libary
Javascript
CSS (vanilla) — co-located stylesheets per component, no framework overhead

The app is available at (https://dev-ready-project.vercel.app/)

Features

Call activity feed — browse all active calls showing direction, phone numbers, call type, date/time, and duration
Call detail view — click any call to see full details including associated notes
Archive calls — remove calls from the activity feed with one click
